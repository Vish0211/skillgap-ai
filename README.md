# SkillGap AI 🚀

An AI-powered career guidance web app that analyzes resumes, identifies skill gaps for a target job role, and generates a personalized 30-day learning roadmap — complete with project suggestions and interview questions.

## Features
- 📄 Resume upload and text extraction (PDF)
- 🎯 Skill-gap analysis across 4 roles: Python Developer, Data Analyst, AI Engineer, Web Developer
- 📅 Auto-generated 30-day learning roadmap
- 💡 Suggested portfolio projects
- 🎯 Sample interview questions
- 🎨 Custom-styled UI

## Tech Stack
- Python
- Streamlit
- pdfplumber (PDF text extraction)

## How to run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
